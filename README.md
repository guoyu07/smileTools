## 工具使用说明

* 快递鸟，需自行去 <http://www.kdniao.com/index.aspx/ /> 注册账号。快递鸟支持全球400家快递公司，接口免费。

    ```
    配置：打开项目至app\handler\KuaiDiQueryHandler.py 配置您的快递鸟

    ```
